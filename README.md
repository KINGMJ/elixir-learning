phoenix安装

```
mix archive.install hex phx_new 1.5.1
```

postgresql安装

```
# 安装
sudo apt-get install postgresql 

# 以 postgres 身份登录控制台
sudo -u postgres psql

# 修改密码
\password postgres
123456
```
