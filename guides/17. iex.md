# 17. iex

使用`Ctrl+C` 退出 iex

使用`h`显示帮助信息，比如`h String.length`

```
iex> h String.length
* def length(string)

  @spec length(t()) :: non_neg_integer()

delegate_to: String.Unicode.length/1

Returns the number of Unicode graphemes in a UTF-8 string.

## Examples

    iex> String.length("elixir")
    6

    iex> String.length("榨崭眨宅斋")
    5
```
