# Elixir 特性


## 语言特性


- 数据结构不可变，immutable


## 语法特性

- Elixir 允许您在调用命名函数时删除括号。这个特性在编写声明和控制流结构时给出了一个更清晰的语法

```
div(10, 2)
div 10, 2
```

- 字符串插值：`#{:world}`

- 字符串连接：`<>`

- 列表`[]`，元组`{}`

- 列表连接/减去：`++/2`和`--/2`

- pin操作符`^`

- 管道操作符 `|>`

- `in`运算符，只能与范围`1..3`和列表`[]`一起使用，可结合`not`操作符一起使用

```
iex> 2 in 1..3
true
iex> 4 not in 1..3
true
```
